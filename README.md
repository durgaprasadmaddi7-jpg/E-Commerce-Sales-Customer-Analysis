# 🛒 E-Commerce Sales & Customer Analysis

## 📌 Project Overview

This project analyzes E-Commerce sales and customer data using MySQL Workbench and Power BI. The objective is to identify business insights such as top-selling products, customer purchasing behavior, revenue trends, and category performance.

---

## 🎯 Objectives

- Analyze sales performance
- Identify top customers
- Find best-selling products
- Analyze category-wise revenue
- Generate business insights using SQL
- Create an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

- SQL
